# Code of Conduct

Bu projede Contributor Covenant 2.1’i temel alıyoruz. Kısa öz:

- Herkese saygılı davran, taciz/ayrımcılık yok.
- Yapıcı geribildirim ver, kişisel saldırı yapma.
- Topluluk alanlarında profesyonel kal.

Kurallara uymayan davranışları repo sahiplerine bildirebilirsin. Detaylar için [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) metnine uyuyoruz.
